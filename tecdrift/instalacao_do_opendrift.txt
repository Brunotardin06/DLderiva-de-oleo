Instalação do OpenDrift em Python  (bkassar..18jul2025)

Baixa o código em https://github.com/OpenDrift/
Vai até o diretório onde descompactou o código
> cd opendrift

Instala o OpenDrift na instalação de Python (cria um ambiente se quiser... eu prefiro instalar globalmente)
> python -m pip install --no-deps -e .

Instala os packages necessários:
> python -m pip install pyproj
> python -m pip install xarray
> python -m pip install cartopy
> python -m pip install netCDF4 
> python -m pip install roaring-landmask
> python -m pip install cmocean
> python -m pip install geojson
> python -m pip install coloredlogs
